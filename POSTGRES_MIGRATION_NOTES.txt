PostgreSQL cleanup notes

- All database helper imports were moved to lib/pg/*.
- External database SDK dependencies were removed from package.json.
- The app now uses DATABASE_URL and the existing pg-based query compatibility layer.
- .env.local.example was cleaned to PostgreSQL/Jira variables only.
- No external database URL/key variables are required.

After extracting:
1) npm install
2) npm run build
3) npm run dev or npm start
